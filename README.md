📋 OMpps｜OM 記錄趴趴寫

手機填重點・回家貼 Word/Excel
給第一線定向行動老師的輕量教學記錄工具


✨ 專案理念

在外奔波上課的定向行動老師，
常常必須在捷運上、候課空檔、咖啡店角落補寫教學記錄。
但 Word / Excel / Google 試算表在手機上操作困難、格子小又容易跑版。

OMpps 的設計理念是：

👉 讓老師用手機「順順填重點」，
👉 回家再貼回 Word / Excel 完成正式文件。

這不是要取代既有表格，而是幫你把「外出補寫記錄」這件事，變成一件舒服又可控的事。


🧩 主要功能
1️⃣ 教學目標模組

逐步填寫教學目標

長期目標可多組

每組長期目標可新增多筆短期目標

2️⃣ 教學記錄模組

可新增第 N 次教學記錄（同一份檔案內）

每次記錄包含：

教學日期

教學時間

教學成效評估（多行）

3️⃣ 匿名草稿代碼（6 位數）

新增記錄時會產生一組 6 位數代碼

用於「繼續未完成」找回草稿

不需帳號、不蒐集個資、極低資安風險

4️⃣ 一鍵下載成單一文字檔

下載 .txt 檔（可直接貼到 Word / Excel）

檔名格式：
YYYYMMDD\_教學記錄\_代碼123456.txt

5️⃣ 日間 / 夜間模式切換 🌞🌙

首頁可選擇白底或黑底模式

記住使用者偏好，長時間填寫更舒適


🔐 資料存放說明

所有草稿暫存在跑 OMpps 的那台電腦本機資料庫（SQLite）

手機只是連線填寫介面，不會儲存資料

按「下載」才會產生 .txt 檔到你的裝置下載資料夾


🚀 快速開始
安裝與啟動

###
python -m venv .venv
..venv\\Scripts\\Activate.ps1
pip install -r requirements.txt
python app.py
###

打開瀏覽器：
http://127.0.0.1:5000

手機同 Wi-Fi 可用電腦內網 IP 存取：
http://192.168.x.x:5000



🧪 使用流程

首頁選擇 ☀️ / 🌙 模式

點選「新增記錄」

記下跳出的 6 位代碼（重要！）

開始填寫教學目標與教學記錄

按「下載」產生 .txt 檔

回家貼到 Word / Excel 完成正式文件
